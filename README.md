# :sushi: EmojiIM: Emoji Input Method for macOS

[![Build Status](https://www.bitrise.io/app/0741e1b8cd1b5086/status.svg?token=79CGJwB83qmYoe5XcqqJbw)](https://www.bitrise.io/app/0741e1b8cd1b5086)

EmojiIM converts github style emoji code to emoji.

![](/docs/sushi.gif)

## :circus_tent: Project goal: Playground for input method
This project goal is not only to prodive emoji input methods, but also to investigate inside of input method. To achive this, some advanced features(e.g. touchbar supports) are implemented and described details at [label:article](https://github.com/mzp/EmojiIM/pulls?utf8=âœ“&q=label%3Aarticle%20).
